# DAILY-WORK-JOURNAL

Marathi-friendly single-page website to track:
- आजचं काम
- किती काम complete झालं (%)
- आजचे देणे / payment details
- उद्याचं schedule

## Run locally

```bash
python3 -m http.server 4173
```

Then open: `http://localhost:4173`

## Publish as permanent link (GitHub Pages)

1. Push this repo to your GitHub account.
2. Go to **Settings → Pages**.
3. Source: **Deploy from a branch**, branch: `main`, folder: `/ (root)`.
4. Your site link will be:
   `https://<your-github-username>.github.io/DAILY-WORK-JOURNAL/`
